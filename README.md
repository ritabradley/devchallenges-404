## Overview

![screenshot](https://res.cloudinary.com/ritalbradley/image/upload/v1600303960/404_screenshot.png)

-   The demo for this challenge can be seen [here](https://placid-detail.surge.sh).
-   My experience with this challenge was pretty pleasant. This was a simple project to brush up on my HTML and CSS skills. I used Tailwind CSS in order to not write any custom CSS, and to make production time shorter since it is a smaller project.
-   While working on this challenge, I got to work on taking a mobile-first approach to responsive development. I normally get projects that are designed at desktop size only and then I'm left to figure out graceful degradation, so this was a nice change.

### Built With

-   [Tailwind](https://tailwindcss.com/)
